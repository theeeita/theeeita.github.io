Пример №2

Установлены пользовательские классы на различные элементы слайдера. Содержимое кнопок также изменено. Слайды запрещено двигать мышкой, также отключено автопроигрывание.

Параметры, которые отличаются от значений по умолчанию (кроме root):

  • autoplay: false
  • draggable: false
  • outerClass: "test-2-wrapper"
  • navClass: "test-2-nav"
  • dotsClass: "test-2-dots"
  • dotItemClass: "test-2_dot-item"
  • buttonsClass: "test-2-buttons"
  • prevButtonHTML: "<i>Назад<i>"
  • nextButtonHTML: "<i>Вперед<i>"
  • prevButtonClass: "test-2-prev"
  • nextButtonClass: "test-2-next"

Пример №3

Слайдер не использует стандартно сгенерированную навигацию. Элементы управления назначены из "внешнего" кода. Также двигать элементы слайдера можно мышкой.

Параметры, которые отличаются от значений по умолчанию (кроме root):

  • nav:false
  • autoplay: false
  • draggable: true
